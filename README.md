# Cyber Security Final Project - Comunication_LTD

This repository contains the final project for our Cyber Security course.

We are building a full-stack web application for a fictional ISP, "Comunication_LTD," with a strong focus on implementing secure development principles and demonstrating web-based vulnerabilities.

## Core Technologies

* Backend: Python (Django)

* Database: MySQL (Relational)

* Frontend: HTML, CSS, JavaScript

* Hashing: PBKDF2, SHA-1

* Security: CSRF Protection, Custom Password Validation, Login Attempt-Limiting, SQL Injection Protection, XSS Protection
